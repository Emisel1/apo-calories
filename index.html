⚙️ Configuration & Accès
📋 Comment ça marche :
1. La clé est stockée uniquement sur ce téléphone, jamais en ligne
2. Chaque entrée met à jour directement le fichier Google Sheets sur Drive
3. Le graphique et les stats se recalculent automatiquement
4. Fonctionne à vie — pas de coût mensuel
🛠️ Comment configurer (une seule fois) :
1. Ouvre Suivi_Apolline_V3 sur Google Sheets
2. Clique Extensions → Apps Script
3. Efface tout et colle le script fourni par Juan
4. Clique Déployer → Nouveau déploiement
5. Type : Application Web
   Exécuter en tant que : Moi
   Accès : Tout le monde
6. Clique Déployer → copie l'URL
7. Colle l'URL dans le champ ci-dessus ✅
💡 Pourquoi Apps Script ? C'est un script qui tourne directement dans ton Google Sheets. Gratuit, permanent, aucune clé API à gérer. Le site lui envoie les données et il les écrit dans le bon onglet automatiquement.
